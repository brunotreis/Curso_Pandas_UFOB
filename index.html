1. Introdução ao Pandas

Pandas é uma biblioteca do Python de código aberto para análise de dados. Ela é uma ferramenta poderosa para a manipulação de dados. Nesse curso, veremos as principais ferramentas dessa biblioteca e como ela pode ser útil para quem trabalha com excel. O pandas pode ser usado em conjunto com outras bibliotecas de processamento numérico, como o Numpy e Scipy, e bibliotecas de visualização de dados como Matplotlib e Plotly. É desejável que o leitor tenha conhecimento de conceitos básicos de Python, como listas, dicionários, etc. A documentação do pandas pode ser encontrada aqui. Usaremos o google colab para rodar nossos códigos.
