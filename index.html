Curso de Pandas - Python

1. Introdução ao Pandas

Pandas é uma biblioteca do Python de código aberto para análise de dados. Ela é uma ferramenta poderosa para a manipulação de dados. Nesse curso, veremos as principais ferramentas dessa biblioteca e como ela pode ser útil para quem trabalha com excel. O pandas pode ser usado em conjunto com outras bibliotecas de processamento numérico, como o Numpy e Scipy, e bibliotecas de visualização de dados como Matplotlib e Plotly. É desejável que o leitor tenha conhecimento de conceitos básicos de Python, como listas, dicionários, etc. A documentação do pandas pode ser encontrada em https://pandas.pydata.org/.

Usaremos a seguinte convenção para importação do pandas:
In [1]: import pandas as pd
Portanto, sempre que aparecer .pd no código, será uma referência ao pandas. Estudaremos duas classes do pandas: Séries e DataFrames. Temos também a opção de importar diretamente
In [2]: from pandas import Series, DataFrame
\

1.1 Séries

Uma série é objeto contendo uma sequência de valores e um array associado de índices. Pense em uma planilha do excel com uma única coluna. Considere a lista contendo cinco números
In [1]: lista = [4, 7, -8, 2,-1]
Podemos transformar essa lista em uma Série
In [2]: obj = pd.Series(lista)
In [3]: obj
Out [3]:
\(\begin{array}{rr} 0 & 4 \\ 1 & 7 \\ 2 & -8 \\ 3 & 2 \\ 4 & -1 \end{array}\)
